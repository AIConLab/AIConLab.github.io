Timeline

  • Aerial robot operation
  • Ground robot familiarization
  • Pupil eye tracking glasses setup
  • Remote operator application walkthrough

Main Task Execution (30 min)

  • Aerial inspection and waypoint creation
  • Ground robot autonomous navigation
  • Gaze-based target identification
  • Post-task survey completion

Task Instructions

Phase 1: Aerial Inspection

  1. Power on drone and controller
  2. Take off from designated landing pad
  3. Fly to inspection target
  4. Create waypoint at target location
  5. Capture clear reference image of target
  6. Return drone to landing pad

Phase 2: Ground Robot Mission

  1. Verify mission files are loaded correctly
  2. Click "Mission Start" to begin autonomous navigation
  3. Monitor ground robot's progress to waypoint
  4. Wait for robot to enter "Inspecting" state

Phase 3: Gaze-Based Inspection

  1. Enable gaze tracking
  2. Look directly at the inspection target on screen
  3. Maintain gaze until fixation point appears (red dot)
  4. Click "Segment" to process target
  5. Review segmentation mask options
  6. Select correct mask using arrow keys
  7. Click "Accept" to confirm selection

Safety Guidelines

Emergency Stop Locations:

  • Remote operator interface
  • Wireless emergency stop controller
  • Robot itself

Important:

  • Maintain awareness of your surroundings
  • Follow all safety instructions from research team
  • Alert staff immediately if you experience any discomfort
