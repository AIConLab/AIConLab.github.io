Description

This test is designed to evaluate the "Gaze Enhanced Multi-robot Inspection System".

In this test an operator (you) will remotely control an aerial robot and direct a ground robot to perform an inspection of an object. We will begin with training on the usage of the aerial robot, ground robot, remote operator application, and Pupil eye tracking glasses.

The test will proceed as follows:

You will fly the drone to the inspection object, set a waypoint, capture a suitable image of the target, then return the drone to the landing pad. The mission files will be loaded to the remote operator application and sent to the ground robot to start the mission. The ground robot will autonomously navigate to the waypoint, scan the scene to localize the inspection object, and approach the inspection object's vicinity.

When the robot has reached the general area of the inspection object, it will enter the Inspecting state and await your instructions. At this point, you will view the robot's camera feed and use your gaze to identify what you would like the robot to inspect.

When the target is identified and successfully segmented, the segmented mask gets sent to the robot to complete the inspection. At this point, your task is completed and the test will be reset. You will then be given a survey to complete after the test task.

Timeline

Equipment Training (30 min)

  • Aerial robot operation
  • Ground robot familiarization
  • Pupil eye tracking glasses setup
  • Remote operator application walkthrough

Main Task Execution (30 min)

  • Aerial inspection and waypoint creation
  • Ground robot autonomous navigation
  • Gaze-based target identification
  • Post-task survey completion

Task Instructions

Phase 1: Aerial Inspection

  1. Power on drone and controller
  2. Take off from designated landing pad
  3. Fly to inspection target
  4. Create waypoint at target location
  5. Capture clear reference image of target
  6. Return drone to landing pad

Phase 2: Ground Robot Mission

  1. Verify mission files are loaded correctly
  2. Click "Mission Start" to begin autonomous navigation
  3. Monitor ground robot's progress to waypoint
  4. Wait for robot to enter "Inspecting" state

Phase 3: Gaze-Based Inspection

  1. Enable gaze tracking
  2. Look directly at the inspection target on screen
  3. Maintain gaze until fixation point appears (red dot)
  4. Click "Segment" to process target
  5. Review segmentation mask options
  6. Select correct mask using arrow keys
  7. Click "Accept" to confirm selection
